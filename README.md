# Arogya-Sahaya Local

Arogya-Sahaya Local is an Android healthcare assistance application designed for elderly and rural users to manage medicines, monitor health vitals, and receive healthcare reminders.

## Features

- Medicine Reminder System
- Vital Health Monitoring
- AI Prescription Simplification
- Emergency SOS Feature
- ASHA Health Camp Information
- 7-Day Health Trend Visualization

## Technologies Used

- Kotlin
- Android Studio
- XML Layouts
- Room Database
- WorkManager
- MPAndroidChart
- Gemini API
- MVVM Architecture

## Purpose

The application aims to improve healthcare accessibility and medicine adherence for elderly users in rural communities.

## Future Scope

- Cloud synchronization
- Real-time healthcare monitoring
- Multi-language support
- Voice-based reminders
